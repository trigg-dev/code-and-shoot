# Code & Shoot 🎮

p5.js でシューティングゲームを作りながらプログラミングを学ぶ教材です。

## 対象

- 小学3年生〜中学1年生
- Scratch 経験者を想定（未経験でもOK）

## はじめかた

### 生徒向け

1. [Releases](../../releases) から最新の `template.zip` をダウンロード
2. デスクトップに展開する
3. `sketch.js` をテキストエディタで開いてコードを書く
4. `index.html` をブラウザで開いて動かす（更新は F5）

### リファレンス

授業で使う p5.js の命令まとめ：  
👉 **[p5.js リファレンス](https://ORG名.github.io/code-and-shoot/reference.html)**

```
code-and-shoot/
```

## カリキュラム概要

| 回 | 内容 | 新しく学ぶこと |
|----|------|----------------|
| 1 | エディタに慣れよう | 座標、関数呼び出し |
| 2 | 変数で絵を動かそう | 変数、変数更新 |
| 3 | キーボードで操作しよう | if文、キー入力 |
| 4 | 画面の端で止まろう | if ネスト |
| 5 | 弾を撃とう | true/false、keyIsDown |
| 6 | 敵を出そう | 複数変数、random() |
| 7 | 弾で敵を倒そう | dist()、当たり判定、スコア |
| 8 | 弾をたくさん撃とう | 配列、push() |
| 9 | 敵をたくさん出そう | for ループ、splice() |
| 10 | 画面をかっこよくしよう | 状態遷移、演出 |
| 11 | AIに動きを考えてもらおう | 関数分離、プロンプティング |
| 12 | 仕上げ＆発表会 | 調整、発表 |

## 使用環境

- ブラウザ（Chrome 推奨）
- テキストエディタ（メモ帳 / サクラエディタ等）
- p5.js v1.9.4
