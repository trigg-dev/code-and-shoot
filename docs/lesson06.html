LESSON 06

弾を撃とう

スペースキーでビームを発射！シューティングゲームっぽくなるよ！

今回のゴール true と false 弾の変数 スペースキーで発射 弾を動かす 画面外で消す 弾を描く デモ

今回のゴール

  • true / false の考え方をおぼえる
  • 変数で「弾が飛んでいるかどうか」を管理する
  • スペースキーで弾を発射できるようにする
  • 画面の外に出た弾を消す
1

true と false（ホントとウソ）

今まで使った変数には数字を入れてきたよね。でも変数には「ホント」か「ウソ」も入れられるんだ。

プログラミングでは、ホントのことを true、ウソのことを false って書くよ。

let bulletActive = false;  // 最初は弾は飛んでいない（ウソ）

この bulletActive が true のときは「弾が飛んでいる」、false のときは「弾がない」ってことになるよ。

💡 true / false のことを「真偽値（しんぎち）」って言うよ。英語では boolean（ブーリアン）。
2

弾の変数を準備しよう

弾を管理するために、4つの変数を追加するよ。

let bulletX = 0;         // 弾のX座標
let bulletY = 0;         // 弾のY座標
let bulletSpeed = 6;     // 弾の速さ
let bulletActive = false; // 弾が飛んでいるか？

bulletX と bulletY は最初は 0 でOK。発射するときに正しい位置をセットするからね。

3

スペースキーで発射！

スペースキーのキーコードは 32。keyIsDown(32) で「スペースキーが押されているか」をチェックできるよ。

でも、弾がすでに飛んでいるときにもう1発撃てたら変になるよね。だから !bulletActive（弾が飛んでいないとき）もチェックするんだ。

if (keyIsDown(32)) {
  if (!bulletActive) {
    bulletActive = true;        // 弾を「飛んでる」にする
    bulletX = x;                  // プレイヤーの位置から発射
    bulletY = y - 20;             // ちょっと上から
  }
}

! は「〜じゃない」っていう意味。!bulletActive は「bulletActive が false のとき」ってことだよ。

💡 !（NOT演算子）は true と false をひっくり返すよ。!true は false、!false は true。
4

弾を動かそう

弾が飛んでいるときだけ、bulletY を減らして上に移動させるよ。

if (bulletActive) {
  bulletY = bulletY - bulletSpeed;  // 上に移動！
}

Y座標を減らすと上に動く、だったよね。bulletSpeed が 6 だから、毎フレーム 6 ピクセルずつ上に飛んでいくよ。

5

画面の外に出たら消す

弾がずっと飛び続けると、画面の外に出ちゃうよね。そうなったら bulletActive を false に戻して、また撃てるようにしよう。

チャレンジ：弾が画面の外に出たら bulletActive を false にしよう！bulletY が 0 より小さくなったら画面外だよ。
💡 ステップ3の発射ロジックを思い出してみよう。bulletActive が false になれば、次にスペースキーを押したときにまた新しい弾を撃てるようになるよ。
6

弾を描こう

弾の描画も、弾があるときだけにしよう。前のレッスンで使った if と ellipse を組み合わせるよ。

チャレンジ：弾があるときだけ描こう！if (bulletActive) を使って ellipse で弾を描いてみよう。色は fill(255, 255, 100) がおすすめ！
弾の色や大きさを変えてみよう！たとえば fill(255, 100, 100) にすると赤い弾になるよ。
