LESSON 04

キーボードで操作しよう

矢印キーで自機を自由に動かそう！

ゴール 1. おさらい 2. if文とは 3. 左右に動かす 4. 上下も挑戦 5. 完成コード デモ

🎯 今回のゴール

キーボードの矢印キー（←→↑↓）で自機を上下左右に動かせるようにする！
「もし〜だったら」の if 文を使うよ。

1

前回のおさらい

前回（Lesson 03）では、変数を使って自機を動かしたね。でも勝手に動くだけで、操作はできなかった。

今回は自分の手で操作できるようにするよ！

let x = 240;
let y = 580;

function setup() {
  createCanvas(480, 640);
}

function draw() {
  background(10, 10, 30);
  fill(60, 180, 255);
  noStroke();
  ellipse(x, y, 36, 44);
}
2

「もし〜だったら」= if 文

if は「もし〜だったら、これをやる」という命令だよ。

日本語で書くとこんな感じ：

「もし 左キーが押されていたら、x を減らす」

これをプログラムで書くと：

if (keyIsDown(LEFT_ARROW)) {
  x = x - 4;
}
💡 keyIsDown() は「今このキーが押されている？」を調べる命令。押されていたら true（はい）、押されてなかったら false（いいえ）を返すよ。
3

左右に動かす

まずは左の移動から。draw() の中に if 文を書こう：

if (keyIsDown(LEFT_ARROW)) {
  x = x - 4;
}

LEFT_ARROW は左矢印キーのこと。左キーが押されている間、x が減って左に動くよ。

右も同じ考え方。今度は x を増やす：

if (keyIsDown(RIGHT_ARROW)) {
  x = x + 4;
}
ここまで書いたら実行してみよう！左右の矢印キーで自機が動くかな？
4

上下も挑戦しよう

左右ができたら、次は上下だ。ここはコードを見ずに自分で書いてみよう！

同じように UP_ARROW と DOWN_ARROW も書いてみよう！上に動かすには y をどうすればいい？下は？
💡 画面の座標では、上にいくほど Y が小さくなるよ。だから上に動かすには y を減らすんだ。ちょっとヘンだけど、慣れると大丈夫！
5

完成コードを確認しよう

上下左右すべて書けたら、こんなコードになるはず：

let x = 240;
let y = 580;

function setup() {
  createCanvas(480, 640);
}

function draw() {
  background(10, 10, 30);

  if (keyIsDown(LEFT_ARROW))  { x = x - 4; }
  if (keyIsDown(RIGHT_ARROW)) { x = x + 4; }
  if (keyIsDown(UP_ARROW))    { y = y - 4; }
  if (keyIsDown(DOWN_ARROW))  { y = y + 4; }

  fill(60, 180, 255);
  noStroke();
  ellipse(x, y, 36, 44);
}
4 の数字を 2 にしたらどうなる？8 にしたら？
自分がちょうどいいと思う速さを見つけてみよう！
発展: 4 を変数にすると便利だよ。let speed = 4; と書いて、x = x - speed; のように使えば、速さをまとめて変えられる！
