LESSON 04

キーボードで操作しよう

矢印キーで自機を自由に動かそう！

ゴール おさらい if文とは 左右に動かす 上下も挑戦 playerSpeed

🎯 今回のゴール

キーボードの矢印キー（←→↑↓）で自機を上下左右に動かせるようにする！
「もし〜だったら」の if 文を使うよ。

1 前回のおさらい

前回（Lesson 03）では、変数を使って自機を動かしたね。でも勝手に動くだけで、操作はできなかった。

今回は自分の手で操作できるようにするよ！

let playerX = 240;
let playerY = 580;

function setup() {
  createCanvas(480, 640);
}

function draw() {
  background(10, 10, 30);
  fill(60, 180, 255);
  noStroke();
  ellipse(playerX, playerY, 36, 44);
}

2 「もし〜だったら」= if 文

if は「もし〜だったら、これをやる」という命令だよ。

日本語で書くとこんな感じ：

「もし 左キーが押されていたら、playerX を減らす」

これをプログラムで書くと：

if (keyIsDown(LEFT_ARROW)) {
  playerX = playerX - playerSpeed;
}
💡 keyIsDown() は「今このキーが押されている？」を調べる命令。押されていたら true（はい）、押されてなかったら false（いいえ）を返すよ。

3 左右に動かす

まずは左の移動から。draw() の中に if 文を書こう：

if (keyIsDown(LEFT_ARROW)) {
  playerX = playerX - playerSpeed;
}

LEFT_ARROW は左矢印キーのこと。左キーが押されている間、playerX が減って左に動くよ。

右も同じ考え方。今度は playerX を増やす：

if (keyIsDown(RIGHT_ARROW)) {
  playerX = playerX + playerSpeed;
}
ここまで書いたら実行してみよう！左右の矢印キーで自機が動くかな？

4 上下も挑戦しよう

左右ができたら、次は上下だ。ここはコードを見ずに自分で書いてみよう！

同じように UP_ARROW と DOWN_ARROW も書いてみよう！上に動かすには playerY をどうすればいい？下は？
💡 画面の座標では、上にいくほど Y が小さくなるよ。だから上に動かすには playerY を減らすんだ。ちょっとヘンだけど、慣れると大丈夫！

5 playerSpeed 変数で速さを管理

今は速さの数字を直接書いているけど、これも変数にしよう。速さを変えたいとき、1か所変えるだけで OK になるよ：

let playerSpeed = 4;

// あとは数字のかわりに playerSpeed を使う
if (keyIsDown(LEFT_ARROW)) {
  playerX = playerX - playerSpeed;
}
playerSpeed を 2 にしたらどうなる？8 にしたら？
自分がちょうどいいと思う速さを見つけてみよう！
