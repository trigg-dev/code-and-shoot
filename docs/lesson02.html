LESSON 02

自機を描こう

シューティングゲームの主人公をキャンバスに登場させよう！

今回のゴール 1. RGBのおさらい 2. キャンバスと背景 3. 自機の色 4. 輪郭線を消す 5. 自機を描く 6. やってみよう デモ

今回のゴール

  • RGBの色指定をおさらいする
  • 自分の機体（じき）をキャンバスに描く
  • noStroke() を使って輪郭線を消す
  • 位置やサイズ、色を自由に変えてみる
1

前回のおさらい ― RGBの色

Lesson 01 で習ったRGBをおぼえてるかな？

おさらい

R（赤）、G（緑）、B（青）の3つの数字（0〜255）で色を作るんだったね。

fill(255, 0, 0) → 赤　fill(0, 255, 0) → 緑　fill(0, 0, 255) → 青

今回はこの色指定を使って、自機（じき）＝プレイヤーの宇宙船を描いていくよ！

2

まずはキャンバスと背景を準備

前回と同じように、setup() でキャンバスを作って、draw() で宇宙っぽい背景を塗ろう。 

createCanvas(480, 640);  // setup() の中に書く
background(10, 10, 30);  // draw() の中に書く

ここまでは Lesson 01 とまったく同じだね。

3

自機の色を決めよう ― fill()

自機は青く光る宇宙船にしよう！ fill(60, 180, 255) で明るい青色を指定するよ。 

fill(60, 180, 255);

この色はどうやって作ってる？

R=60（赤はちょっとだけ）、G=180（緑はそこそこ）、B=255（青は最大！）
青をメインに、緑もまぜることで水色っぽい明るい青になるんだ。

4

輪郭線を消そう ― noStroke()

ふつうに図形を描くと、まわりに黒い線（輪郭線＝ストローク）がつくよね。 noStroke() を書くと、その線を消せるんだ。 

noStroke();

ゲームのキャラは輪郭線がないほうがキレイに見えることが多いよ。

5

自機を描こう ― ellipse()

ellipse(x, y, 横はば, 縦はば) で楕円を描くよ。自機の位置は画面の下のほう、まんなかあたりにしよう。

位置とサイズの意味

ellipse(240, 580, 36, 44) は…
・X = 240 → 横の位置（キャンバス幅480の半分 ＝ まんなか）
・Y = 580 → 縦の位置（下のほう。キャンバス高さは640）
・横はば = 36 → 横幅
・縦はば = 44 → 縦幅（横より大きいので、たて長の楕円になるよ） 

ellipse(240, 580, 36, 44);
6

やってみよう

ここまでのコードを draw() の中にまとめて書いてみよう。宇宙の暗い背景に、青く光る自機が浮かんでいるはず！

位置を変えてみよう

ellipse の最初の2つの数字（XとY）を変えてみよう。自機が動くよ！

例：Xを 100 にしたら左のほう、Yを 400 にしたら上のほうに行くかな？

サイズを変えてみよう

ellipse の後ろの2つの数字（横はばと縦はば）を変えてみよう。大きくしたり小さくしたり、同じ数字にしたらどうなる？

色を変えてみよう

fill の数字を変えて、自分だけの機体カラーにしよう！

例：fill(255, 100, 50) にしたらどんな色？ fill(255, 255, 0) は？
