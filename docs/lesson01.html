1

setup() と draw() ― プログラムの2本柱

p5.js のプログラムには、大事な関数（きのう）が2つあるよ。

setup()（セットアップ）

プログラムが始まるとき、最初に1回だけ動く。準備をする場所だよ。

draw()（ドロー）

setup() が終わったら、ずーっと何回もくりかえし動く。ここに「描きたいもの」を書くよ。1秒間に約60回も動いてるんだ！

function setup() {
  // ここは最初に1回だけ動く
}

function draw() {
  // ここはずっとくりかえし動く
}
2

キャンバスを作ろう ― createCanvas()

まずは「おえかきするスペース」を用意しよう。createCanvas(横, 縦) で大きさを決めるよ。 

function setup() {
  createCanvas(480, 640);
}

これで横480ピクセル、縦640ピクセルのキャンバスができるよ。

XY座標ってなに？

キャンバスの位置は X（よこ）と Y（たて）の数字で決まるよ。

(0, 0) ←左上がスタート地点！
→ X が大きくなると 右へいく
↓ Y が大きくなると 下へいく

右下のすみっこは (480, 640) だよ

数学の座標とちがって、下にいくほどYが大きくなるのがポイント！

3

背景を塗ろう ― background()

background(R, G, B) で背景を塗れるよ。R（赤）、G（緑）、B（青）の3つの数字（0〜255）で色を作るんだ。

RGBってなに？

光の三原色 赤(R) 緑(G) 青(B) をまぜて色を作るしくみ。それぞれ 0（なし）〜 255（最大）の数字で決めるよ。

function setup() {
  createCanvas(480, 640);
}

function draw() {
  background(10, 10, 30);
}

background(10, 10, 30) は暗い青っぽい色。宇宙みたいでカッコいいでしょ？

色をぬろう ― fill()

fill(R, G, B) は、これから描く図形の色を決めるよ。background と同じで RGB で指定するんだ。 

fill(255, 0, 0);  // 赤！
🤔 クイズ！

fill(255, 255, 255) にしたら何色になる？

fill(0, 0, 0) は？

ヒント：全部最大＝光が全部まざる ／ 全部0＝光がない

5

図形を描こう ― rect() と ellipse()

いよいよ図形を描くよ！

rect(x, y, 横はば, 縦はば)

四角形を描く。(x, y) が左上のかどの位置だよ。

fill(255, 0, 0);
rect(190, 270, 100, 100);

ellipse(x, y, 横はば, 縦はば)

楕円（だえん）・丸を描く。(x, y) が中心の位置だよ。

同じように ellipse(x, y, 横はば, 縦はば) で丸も描いてみよう！
