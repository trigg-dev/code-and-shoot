LESSON 08

弾で敵を倒そう

当たり判定とスコアでゲームらしくなるよ！

今回のゴール 1. 当たり判定って？ 2. dist() で距離 3. 当たったら倒す 4. スコアをつける 5. やってみよう 発展チャレンジ デモ

今回のゴール

  • dist() で2つの丸の距離をはかる
  • 弾が敵に当たったら敵を倒す「当たり判定」を作る
  • スコアを画面に表示する
1

弾が当たったらどうする？

今のゲーム、弾を撃っても敵をすり抜けちゃうよね。シューティングゲームなのに倒せないなんて！

弾が敵に「当たった」ことをプログラムに教えてあげないといけない。でも、どうやって？

「当たった」ってどういうこと？

弾も敵も丸（ellipse）で描いてるよね。2つの丸が近づいて重なったら「当たった」と考えるよ。

つまり、弾の位置と敵の位置が近いかどうかを調べればいいんだ！

2

dist() で距離をはかろう

p5.js には dist() という便利な命令があるよ。2つの点がどれくらい離れているかを計算してくれる。

let d = dist(bulletX, bulletY, enemyX, enemyY);

これで変数 d に「弾と敵の間の距離」が入る。d が小さいほど近いってことだね。

弾の半径はだいたい 3、敵の半径は 18。合わせると 21 だけど、ざっくり d < 20 で「当たった！」と判定しよう。

💡 dist(x1, y1, x2, y2) は、点(x1, y1) と点(x2, y2) の直線距離を返すよ。数学でいう「ピタゴラスの定理」を勝手に計算してくれるんだ。
3

当たったら敵を倒そう

距離がはかれたら、あとは if で「近かったら何かする」を書くだけ！

当たったときにやることは2つ：

  • 弾を消す → bulletActive = false
  • 敵をリスポーン → 前回と同じ、上に戻してランダム位置
if (bulletActive && enemyActive) {
  let d = dist(bulletX, bulletY, enemyX, enemyY);
  if (d < 20) {
    bulletActive = false;
    enemyX = random(20, 460);
    enemyY = -20;
    enemySpeed = random(1, 4);
  }
}

これを draw() の中、敵の移動処理のあとに書こう。

💡 && は「〜かつ〜」の意味。弾が飛んでいて、かつ敵がいるときだけ判定するよ。両方チェックすることで、無駄な計算をしなくて済むんだ。
4

スコアをつけよう

敵を倒したら、スコアを増やそう！まず変数を追加：

let score = 0;

そして、ステップ3の当たり判定の中に score = score + 1; を追加するよ。

スコアを画面に表示するには text() を使おう。描画処理の一番最後に書いてね：

fill(255);
textSize(18);
text("SCORE: " + score, 10, 30);

text() の使い方

text("表示したいもの", X, Y) で画面に文字を出すよ。

"SCORE: " + score で、文字と変数の値をつなげて表示できる。スコアが 5 なら SCORE: 5 と出るよ。

5

やってみよう

ここまで書けたら実行してみよう！弾が敵に当たったらスコアが増えるはず。

d < 20 の 20 を変えてみよう！50 にしたら当てやすくなるかな？10 にしたら？
スコアの色を変えてみよう！fill(255, 255, 100) にしたら黄色になるよ。位置も変えてみて。
textSize() の数字を大きくしてみよう。32 にしたらどうなる？
★

発展チャレンジ：敵にぶつかったら？

dist() は弾と敵の距離だけじゃなく、プレイヤーと敵の距離もはかれるよ。

もしプレイヤーが敵にぶつかったら、何か起きるようにしてみよう！

dist(x, y, enemyX, enemyY) でプレイヤーと敵の距離を調べて、近すぎたら何かしてみよう！
💡 ヒント：ぶつかったら score = 0; でスコアリセット、とか background(255, 0, 0); で画面を赤くフラッシュ、とか。いろいろ実験してみて！
