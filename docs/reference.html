p5.js リファレンス

Code & Shoot で使う命令まとめ

きほん 図形をかく 色をぬる 変数 もし〜なら キー入力 計算・ランダム 文字を出す 配列 くりかえし 関数をつくる

🚀 きほん

createCanvas(よこ, たて)
画面（キャンバス）をつくる。setup() の中で1回だけ使う。
引数意味
よこ画面の横の大きさ（ピクセル）
たて画面のたての大きさ（ピクセル）
function setup() {
  createCanvas(480, 640);
}
setup()
プログラムが始まるとき1回だけ動く場所。画面サイズなど、最初の準備を書く。
function setup() {
  createCanvas(480, 640);
}
Scratchでいうと「🏁旗がクリックされたとき」と同じ！
draw()
1秒に約60回、ずっとくりかえし動く場所。ゲームのメイン処理を書く。
function draw() {
  background(10, 10, 30);
  ellipse(240, 300, 50);
}
Scratchでいうと「ずっと」ブロックと同じ！
background(r, g, b)
画面ぜんたいを塗りつぶす。draw() の最初に書くと、前のフレームを消してくれる。
引数意味
r赤の強さ（0〜255）
g緑の強さ（0〜255）
b青の強さ（0〜255）
background(10, 10, 30);   // 暗い宇宙っぽい色
background(255, 0, 0);    // 真っ赤
background(0);             // 真っ黒（1つだけでもOK）

⭕ 図形をかく

ellipse(x, y, よこ幅, たて幅)
丸（楕円）をかく。x, y は丸の中心の位置。
引数意味
x中心の横の位置
y中心のたての位置
よこ幅横の大きさ
たて幅たての大きさ（省略するとまん丸）
ellipse(240, 300, 50);        // まん丸（直径50）
ellipse(240, 300, 60, 40);    // 横長の楕円
rect(x, y, よこ幅, たて幅)
四角をかく。x, y は左上の角の位置。
引数意味
x左上の横の位置
y左上のたての位置
よこ幅横の大きさ
たて幅たての大きさ
rect(100, 200, 80, 60);   // 横80×たて60の四角
triangle(x1, y1, x2, y2, x3, y3) 応用
三角形をかく。3つの角の位置（x, y）を指定する。
引数意味
x1, y11つ目の角の位置
x2, y22つ目の角の位置
x3, y33つ目の角の位置
triangle(240,100, 200,200, 280,200); // 上向き三角
line(x1, y1, x2, y2)
2つの点をつなぐ直線をひく。
line(0, 0, 480, 640);   // 左上から右下へ線をひく

🎨 色をぬる

fill(r, g, b)
これより後に描く図形の中身の色を決める。
fill(60, 180, 255);    // 水色
ellipse(240, 300, 50); // この丸が水色になる

fill(255, 255, 100);   // 黄色
ellipse(100, 300, 30); // この丸は黄色になる
fill は「塗る」という意味。図形の「前」に書くのがポイント！
noStroke()
図形のフチの線を消す。
noStroke();              // フチなし
ellipse(240, 300, 50); // フチのない丸
stroke(r, g, b)
図形のフチの色を決める。
stroke(255, 0, 0);     // フチを赤に

📍 座標のしくみ

p5.js の画面は左上が (0, 0)。
右にいくほど x が増える。
下にいくほど y が増える（Scratchとは逆！）。 
(0, 0) ─────────→ x が増える (480, 0)
  │
  │
  │    (240, 320) ← 画面のまんなか
  │
  ↓
 y が増える
(0, 640)                         (480, 640)
Scratchでは上にいくとyが増えるけど、p5.jsは逆！ 下にいくとyが増えるよ。

📦 変数（へんすう）

let 名前 = 値;
データを入れる箱をつくる。名前をつけて、中に値を入れる。
let playerX = 240;    // playerXという箱に240を入れる
let speed = 4;         // speedという箱に4を入れる
let alive = true;      // aliveという箱にtrueを入れる
Scratchの「変数を作る」と同じ！ = は「等しい」じゃなくて「入れる」という意味だよ。
変数の更新
箱の中身を書きかえる。draw() の中で書きかえると、毎フレーム変わるから動きになる！
playerX = playerX + 1;   // 右に1ずつ動く
playerX = playerX - 3;   // 左に3ずつ動く
score = score + 1;       // スコアを1増やす
true と false
「はい」か「いいえ」だけの特別な値。スイッチのON/OFFみたいなもの。
let bulletActive = false;  // 最初は弾が出ていない

// 発射するとき
bulletActive = true;           // 弾が出ている状態にする

// 画面の外に出たら
bulletActive = false;          // 弾を消す

🔀 もし〜なら（if文）

if (条件) { }
条件が正しいときだけ、{ } の中が動く。
if (score > 100) {
  // スコアが100より大きいときだけ動く
}
Scratchの「もし〜なら」ブロックと同じ！
くらべる記号
if の中で使う記号まとめ。
記号意味れい
===同じscore === 0
!==同じじゃないstate !== "gameover"
>より大きいplayerX > 460
<より小さいbulletY < 0
>=以上speed >= 5
<=以下playerX <= 20
= は「入れる」、=== は「同じかくらべる」。まちがえやすいから注意！
if の中に if（ネスト）
条件の中にさらに条件を書ける。「〜して、さらに〜なら」。
if (keyIsDown(LEFT_ARROW)) {
  if (playerX > 20) {
    playerX = playerX - speed;
  }
}
「左キーが押されていて、さらに左端じゃなければ動く」という意味！
! （ひっくり返す）
! をつけると、true と false がひっくり返る。
if (!bulletActive) {
  // bulletActive が false のときだけ動く
  // つまり「弾が出てないときだけ」
}

⌨️ キー入力

keyIsDown(キーの番号)
キーが押されている間ずっと true になる。draw() の中で使う。
キー書き方
← 左keyIsDown(LEFT_ARROW)
→ 右keyIsDown(RIGHT_ARROW)
↑ 上keyIsDown(UP_ARROW)
↓ 下keyIsDown(DOWN_ARROW)
スペースkeyIsDown(32)
if (keyIsDown(LEFT_ARROW)) {
  playerX = playerX - speed;
}
if (keyIsDown(32)) {
  // スペースキーが押されている
}
移動にも発射にも使える万能な命令！

🎲 計算・ランダム

random(最小, 最大)
最小〜最大の間のランダムな数字を返す。サイコロみたいなもの。
let x = random(0, 480);      // 0〜480のどこか
let size = random(20, 50);    // 20〜50のどこか
let speed = random(2, 8);     // 2〜8のどこか
dist(x1, y1, x2, y2)
2つの点がどれくらい離れているかを計算する。当たり判定で使う！
let d = dist(playerX, playerY, enemyX, enemyY);
if (d < 30) {
  // 30ピクセルより近い → ぶつかった！
}
2つの丸が近すぎたら「当たった」と判定するよ。
frameCount
プログラムが始まってから何フレーム経ったかの数字。勝手に増えていく。
// 60フレームごとに何かする（約1秒ごと）
if (frameCount % 60 === 0) {
  // 1秒たった！
}
% は「割った余り」。60で割って余りが0 → ちょうど60の倍数のとき。

✏️ 文字を出す

text(内容, x, y)
画面に文字や数字を表示する。
text("GAME OVER", 240, 300);
text("スコア: " + score, 10, 30);
textSize(大きさ)
文字の大きさを変える。
textSize(32);   // 大きい文字
textSize(14);   // 小さい文字
textAlign(横, たて)
文字の位置ぞろえを決める。
textAlign(CENTER, CENTER);   // ど真ん中ぞろえ
textAlign(LEFT, TOP);         // 左上ぞろえ

🗄️ 配列（はいれつ）中級

let 名前 = [];
データをたくさん入れられるリストをつくる。ロッカーの棚みたいなもの。
let bullets = [];   // 空のリストをつくる
配列.push(データ)
リストの最後にデータを追加する。
bullets.push({
  x: playerX,
  y: playerY,
  speed: 6
});
// リストに弾丸1つ分のデータを追加
配列[番号]
リストの中身を番号で取り出す。番号は0から始まる。
bullets[0]       // リストの1番目
bullets[0].x     // 1番目の弾のx座標
bullets.length   // リストに何個入ってるか
配列.splice(番号, 個数)
リストから要素を引っこ抜く。
bullets.splice(i, 1);  // i番目を1個抜く
前からぬくと番号がズレるから、後ろからやると安全だよ。
{ } オブジェクト
いくつかのデータを1つのかたまりにまとめる。配列に入れるときに便利。
// 弾丸1つ分のデータ
let bullet = {
  x: 240,
  y: 580,
  speed: 6
};

bullet.x      // 240
bullet.speed  // 6
データに名前をつけてまとめるイメージ。名札みたいなもの！

🔄 くりかえし（for ループ）中級

for (let i = 0; i < 回数; i++)
配列の全部に同じ処理をするときに使う。
for (let i = 0; i < bullets.length; i++) {
  let b = bullets[i];
  b.y = b.y - b.speed;         // 動かす
  ellipse(b.x, b.y, 6, 12);   // 描く
}
i は「今何番目を見てるか」の番号。0から始まって1ずつ増える。
後ろからループ（消すとき用）
配列から要素を消すときは後ろからループすると安全。
for (let i = bullets.length - 1; i >= 0; i--) {
  let b = bullets[i];
  if (b.y < 0) {
    bullets.splice(i, 1);  // 画面外の弾を消す
  }
}
前から消すと番号がズレちゃう。後ろからなら安全！

🧩 関数をつくる

function 名前() { }
処理をまとめて名前をつける。同じ処理を何度も書かなくてよくなる。
// 関数をつくる
function drawPlayer() {
  fill(60, 180, 255);
  ellipse(playerX, playerY, 36, 44);
}

// draw() の中で使う
function draw() {
  background(10, 10, 30);
  drawPlayer();   // 呼ぶだけでOK！
}
Scratchの「ブロック定義」と同じ！ コードが整理されて読みやすくなるよ。
function 名前(引数) { }
関数にデータを渡すことができる。引数（ひきすう）= 渡すデータ。
function drawBullet(x, y) {
  fill(255, 255, 100);
  ellipse(x, y, 6, 12);
}

// 使うとき：位置を渡す
drawBullet(240, 300);
drawBullet(100, 500);
x, y は「仮の名前」。呼ぶときに実際の数字を入れると、その数字が使われる。
