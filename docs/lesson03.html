LESSON 03

変数で動かそう

「名前をつけた箱」を使って、自機を動かしてみよう！

今回のゴール 前回のおさらい 変数ってなに？ 変数で自機を描く 動かしてみよう 実験してみよう デモ

🎯 今回のゴール

「変数」を使って、自機が画面の中を動くようにする！
変数は「名前をつけた箱」。数字を入れておいて、あとから変えられるよ。

1

前回のおさらい

前回（Lesson 02）では、画面に自機を描いたよね。

自機は画面の 240, 580 にいて、ぜんぜん動かない。今回はこれを動くようにするよ！

2

変数ってなに？

変数（へんすう）は「名前をつけた箱」だよ。

箱に数字を入れておいて、あとで取り出したり、中身を変えたりできる。

📦 箱に名前をつけて、数字を入れる：

let x = 240;

let は「箱をつくるよ！」という合図。
x が箱の名前。
240 が箱に入れた数字。

💡 x は「プレイヤーのX座標（横の位置）」という意味の名前だよ。好きな名前をつけられるけど、わかりやすい名前がいいね！
3

変数を使って自機を描く

数字の 240 を直接書くかわりに、変数を使ってみよう。ellipse の中で変数名を使うだけ：

ellipse(x, y, 36, 44);

これだけだと、前回と同じで自機は止まったまま。でも準備は OK！

4

変数の中身を変える → 動く！

ここがポイント！draw() の中で、毎フレーム x に 1 を足してみよう：

x = x + 1;  // 毎フレーム右に1ピクセル動く！

draw() は1秒に約60回動くから、毎回 x が 1 ずつ増えて、自機がどんどん右に動いていくよ！

💡 x = x + 1 は「今の x に 1 を足して、また x に入れ直す」という意味。箱の中身を取り出して、1を足して、また箱に戻すイメージだね。
5

数字を変えて実験してみよう

+ 1 の部分を変えると、動き方が変わるよ：

x = x + 1 を x = x + 3 にしたらどうなる？
-1 にしたら？
y を変えてみたら？

自機が右にスーッと動いていくよ。画面の外に出たらまた左から出てくるよ。
